# 🚀 Motivation School Diary

**Motivation School Diary** — это современный электронный школьный дневник с элементами геймификации. Проект создан для повышения мотивации учеников через систему баллов и обеспечения удобного взаимодействия между школой, родителями и детьми.

## ✨ Основные функции

### 🧑‍🎓 Кабинет ученика
- **Система баллов:** Получение вознаграждений за активность и оценки.
- **Прогресс-бар:** Визуализация достижений и уровней.
- **Расписание и ДЗ:** Удобный доступ к школьному расписанию и деталям домашних заданий.

### 👨‍👩‍👧 Кабинет родителя
- **Графики успеваемости:** Визуализация прогресса ребенка с помощью Chart.js.
- **Система поощрений:** Возможность отправить виртуальный «бонус» ребенку.

### 👩‍🏫 Кабинет учителя
- **Статистика класса:** Просмотр списка учеников, их текущих оценок и баллов.
- **Управление заданиями:** Интерфейс для добавления новых домашних работ.

### 🔐 Авторизация и персонализация
- Выбор региона и конкретной школы при регистрации.
- Сохранение данных профиля и прогресса в `localStorage` (данные не пропадают после перезагрузки страницы).

## 🛠 Технологии

- **Frontend:** Чистый HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+).
- **Графики:** Chart.js.
- **Шрифты:** Inter.
- **Хранение данных:** `localStorage` (браузерное хранилище) и статический `db.json`.

## 🚀 Как запустить проект локально

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/Pidoraska228/school-motivation.git
   ```
2. Перейдите в папку проекта.
3. Откройте файл `index.html` в любом современном браузере.

> **Важно:** Для корректной загрузки данных из `db.json` рекомендуется использовать локальный сервер (например, расширение **Live Server** в VS Code).
